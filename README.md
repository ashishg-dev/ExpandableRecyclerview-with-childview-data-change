# ExpandableRecyclerview-with-childview-data-change (Kotlin)
Working with Parent-Child Recyclerview and update child view details.

<h2>Features Implemented</h2><br/>

1 - Picasoo.<br/>
2 - Recyclerview.<br/>
3 - Custom Adapter with child recyclerview.<br/>
4 - InputDialog.<br/>
